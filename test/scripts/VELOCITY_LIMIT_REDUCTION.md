# 降低Tube MPC和Safe-Regret MPC线速度限制

## 修改日期
2026-04-06

## 修改目标
降低两个求解器的**线速度**上限和参考速度，以提高系统安全性和稳定性。**角速度保持不变**以维持快速转向能力。

## 修改详情

### Tube MPC 参数文件
**文件**: `src/tube_mpc_ros/mpc_ros/params/tube_mpc_params.yaml`

| 参数 | 原值 | 新值 | 降低幅度 | 说明 |
|------|------|------|---------|------|
| `max_vel_x` | 1.5 | 0.8 | -47% | 最大线速度 |
| `mpc_ref_vel` | 0.8 | 0.5 | -37% | MPC参考速度 |
| `max_speed` | 1.0 | 0.6 | -40% | 最大速度 |
| `max_rot_vel` | 3.5 | 3.5 | 0% | 最大旋转速度（保持不变） |
| `mpc_max_angvel` | 3.0 | 3.0 | 0% | MPC最大角速度（保持不变） |

### Safe-Regret MPC 参数文件
**文件**: `src/safe_regret_mpc/params/safe_regret_mpc_params.yaml`

| 参数 | 原值 | 新值 | 变化幅度 | 说明 |
|------|------|------|---------|------|
| `ref_vel` | 0.5 | 0.3 | -40% | 参考速度 |
| `u_max[0]` | 1.0 | 0.6 | -40% | 输入线速度上限 |
| `v_max` (terminal_set) | 1.0 | 0.6 | -40% | 终端集最大速度 |
| `max_angvel` | 3.5 | 5.0 | +43% | 最大角速度（已增加） |
| `u_max[1]` | 1.5 | 2.5 | +67% | 输入角速度上限（已增加） |

## 修改效果

### 预期改进

1. **提高安全性**：
   - 更低的线速度意味着更快的反应时间
   - 减少撞墙时的冲击力
   - 更容易进行紧急制动

2. **提高机动性**：
   - 更高的角速度允许更快的转向
   - 在狭窄空间中更灵活
   - 快速避障能力增强

3. **提高稳定性**：
   - 减少MPC求解器的数值问题
   - 降低线速度动态复杂性
   - 减少因速度过快导致的控制不稳定

4. **减少崩溃**：
   - safe_regret_mpc_node崩溃率降低
   - 约束不可行情况减少
   - 数值计算更稳定

### 可能的缺点

1. **任务完成时间增加**：
   - 平均线速度降低约40%
   - 相同距离的导航时间增加
   - 测试总时间延长

2. **角速度过快可能导致不稳定**：
   - 高速转向可能引起打滑
   - 可能需要调整转向权重参数

## 数值对比

### 典型导航任务时间估算

假设导航距离：15米（从起点到取货点再到卸货点）

**修改前**：
- 平均速度：0.8 m/s
- 预计时间：15 / 0.8 ≈ 19秒

**修改后**：
- 平均速度：0.5 m/s（考虑实际速度约为上限的80%）
- 预计时间：15 / 0.5 ≈ 30秒

**增加时间**：约11秒（+58%）

**注意**：角速度保持不变，所以转向时间不会增加，实际增加的时间可能少于估算值。

## 速度限制总结

### 线速度限制（已降低）

| 组件 | 限制 | 说明 |
|------|------|------|
| Tube MPC max_vel_x | 0.8 m/s | 最大线速度（降低47%） |
| Tube MPC mpc_ref_vel | 0.5 m/s | 参考速度（降低37%） |
| Safe-Regret MPC ref_vel | 0.3 m/s | 参考速度（降低40%） |
| Safe-Regret MPC u_max[0] | 0.6 m/s | 输入上限（降低40%） |
| Safe-Regret MPC terminal v_max | 0.6 m/s | 终端集上限（降低40%） |

**有效线速度上限**：0.6 m/s (以Safe-Regret MPC的u_max为准)

### 角速度限制

| 组件 | 限制 | 说明 |
|------|------|------|
| Tube MPC max_rot_vel | 3.5 rad/s (~200°/s) | 最大角速度（保持不变） |
| Tube MPC mpc_max_angvel | 3.0 rad/s (~172°/s) | MPC角速度（保持不变） |
| Safe-Regret MPC max_angvel | 5.0 rad/s (~286°/s) | 最大角速度（增加43%） |
| Safe-Regret MPC u_max[1] | 2.5 rad/s (~143°/s) | 输入上限（增加67%） |

**有效角速度上限**：2.5 rad/s (~143°/s) (以Safe-Regret MPC的u_max为准)

### 验证修改效果

1. **运行测试**：
   ```bash
   ./test/scripts/run_automated_test.py --model tube_mpc --shelves 1 --no-viz
   ./test/scripts/run_automated_test.py --model safe_regret --shelves 1 --no-viz
   ```

2. **观察指标**：
   - 平均速度（从metrics.json）
   - 任务完成时间
   - 撞墙次数
   - 节点崩溃次数

3. **RViz观察**：
   - 机器人运动是否更平滑
   - 转向是否更稳定
   - 是否还有突然加速

### 如果需要进一步调整

**如果线速度还是太快**：
- 继续降低 `ref_vel` (建议降到0.2)
- 降低 `u_max[0]` (建议降到0.4)

**如果线速度太慢**：
- 提高 `ref_vel` (建议提高到0.4)
- 提高 `u_max[0]` (建议提高到0.8)

**如果角速度需要调整**（当前已增加）：
- 提高 `max_angvel` (建议提高到6.0)
- 降低 `max_angvel` (建议降低到2.0)
- 提高 `u_max[1]` (建议提高到3.0)
- 降低 `u_max[1]` (建议降低到1.5)
