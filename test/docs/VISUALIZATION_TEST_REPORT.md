## 问题解决记录

### 问题1：Pandas兼容性

**错误：**
```
ValueError: Multi-dimensional indexing no longer supported
```

**解决：**
- 创建简化分析脚本
- 使用`.values`转换pandas Series为numpy数组

**状态：** ✅ 已解决

### 问题2：Goal数据缺失

**现象：**
```python
goal_x = df['goal_x'].iloc[0]  # Returns NaN
```

**原因：**
- 目标发布器启动时间晚于数据记录器
- 或目标话题未正确连接

**临时解决方案：**
- 在图表中跳过goal标记

**长期解决方案：**
- 优化启动顺序
- 添加topic同步机制

## 推荐使用流程

### 标准测试流程

```bash
# 1. 运行测试（自动记录数据）
roslaunch tube_mpc_benchmark visualization_test.launch \
    goal_x:=5.0 goal_y:=5.0 \
    enable_recorder:=true

# 2. 等待测试完成（~60秒）

# 3. 分析数据
python3 simple_analyze.py \
    /tmp/tube_mpc_data/navigation_data_*.csv \
    my_analysis.png

# 4. 查看结果
eog my_analysis.png
```

### 批量分析流程

```python
import glob
import subprocess

# 分析所有测试数据
csv_files = glob.glob('/tmp/tube_mpc_data/*.csv')
for csv_file in csv_files:
    output = csv_file.replace('.csv', '_analysis.png')
    subprocess.run(['python3', 'simple_analyze.py', csv_file, output])
```
