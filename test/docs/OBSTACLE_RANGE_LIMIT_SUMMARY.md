# 随机障碍物范围限制 - 修改说明

**日期**: 2026-04-05
**修改**: 限制障碍物随机刷新范围在地图中间区域

---

## 📋 修改内容

### 障碍物随机范围
| 坐标轴 | 修改前 | 修改后 |
|--------|--------|--------|
| **X坐标** | [-10, 10] | **[-4, 4]** ✅ |
| **Y坐标** | [-10, 10] | [-10, 10] (不变) |

### 代码修改
```python
# 修改前
x = random.uniform(self.map_min_x + obstacle_size/2, self.map_max_x - obstacle_size/2)
# X ∈ [-10, 10]

# 修改后
x = random.uniform(-4.0, 4.0)
# X ∈ [-4, 4]
```

## 🎯 效果说明

### 地图布局
```
     ┌─────────────────────────────────────┐
     │                                     │
 10  │  (障碍物不会出现在这里)              │
     │                                     │
     │  ┌─────────────────────┐            │
  4   │  │   障碍物随机区域     │ ← X=4      │
     │  │   X ∈ [-4, 4]      │            │
     │  │   Y ∈ [-10, 10]    │            │
  0   │  │  ● ● ●            │            │
     │  │                   │            │
 -4  │  │                   │ ← X=-4     │
     │  └─────────────────────┘            │
     │                                     │
-10  │  (障碍物不会出现在这里)              │
     │                                     │
     └─────────────────────────────────────┘
      -10   -8   -6   -4   -2   0   2   4   6   8  10
```

### 行为说明
1. **启动时**：障碍物在X∈[-4, 4], Y∈[-10, 10]范围内随机放置
2. **避开小车**：不会出现在小车初始位置(-8, 0)周围1.5m范围内
3. **避开目标**：不会出现在目标点(8, 0)周围0.6m范围内
4. **固定位置**：整个测试过程中位置不变
