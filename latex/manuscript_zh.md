# 随机部分可观测机器人时序逻辑任务的安全遗憾模型预测控制

## 摘要

本文提出了安全遗憾模型预测控制，这是一个统一的框架，集成了时序逻辑任务规范、概率安全性以及针对不确定性和部分可观测性下安全关键机器人控制的学习保证。该方法在约束相对于安全可行策略类的遗憾的同时优化任务性能，从而在闭环中强制执行任务合规性和风险限制。本文：(i) 形式化了针对以至少 $1-\delta$ 概率满足信号时序逻辑（STL）规范的所有策略的约束（安全）遗憾概念；(ii) 使用可微鲁棒性代理将STL提升到置信空间，使得可以在MPC中可处理地纳入逻辑目标；(iii) 通过分布鲁棒机会约束执行安全性，并使用在线数据驱动的紧缩来考虑有限样本误差和分布偏移；(iv) 通过将参考学习与管道/跟踪MPC耦合，将无遗憾保证"下沉"到动态可行层，证明在有界跟踪误差下，抽象层的 $o(T)$ 遗憾会产生 $o(T)$ 的动态遗憾。本文进一步构造了分布鲁棒控制不变终端集、复合Lyapunov-Barrier条件和鲁棒性预算更新，以在滚动时域中同时建立递归可行性、类ISS稳定性和概率满足下界。在协作装配和负载运输任务上的实验表明，更高的STL满足概率、减少的安全/动态遗憾以及实时执行，消融研究确认了每个组件的必要性。结果为安全关键机器人系统提供了整合逻辑级目标、不确定性和具有可证明保证的学习的原则性途径。

## 关键词

机器人安全；模型预测控制；时序逻辑；机会约束优化；无遗憾学习

## 1. 引言

机器人越来越多地在需要在单个反馈架构中解决任务级指令、物理可行性和不确定环境下安全性的环境中运行。时序逻辑（如信号时序逻辑（Signal Temporal Logic，STL）和线性时序逻辑（Linear Temporal Logic，LTL））通过机器可验证的语义捕获截止时间、顺序和持久性，而模型预测控制（Model Predictive Control，MPC）作为优化满足动力学和约束的性能的原则性工具。连接这些层次需要的方法能够：(i) 在考虑不确定性的反馈水平上表达逻辑目标，(ii) 在分布偏移下保证概率安全性，以及 (iii) 在控制时间尺度上保持动态可行性。过去五年在每个轴上都取得了实质性进展，然而，一个端到端的滚动时域解决方案，同时量化相对于安全可行基线的闭环性能，仍然难以捉摸。

将STL嵌入MPC已从离线综合发展到优化规范鲁棒性概念的闭环公式。近期工作形式化了滚动时域中的监控，并展示了鲁棒性分数如何在不违反语义的情况下指导在线优化[1]。嵌套STL的连续时间综合提高了对于交织到达-避免和停留时间要求的机器人任务的表达能力[2]。同时，自触发实现以频繁重新规划换取证书支持的事件间更新，减少计算同时保持满足[3]。学习也进入了循环：神经MPC变体优化STL鲁棒性的可微代理，用于操作和导航基准[4]，基于采样的规划器使用STL项塑造轨迹成本以加速探索，同时保持规范感知[5]。这些发展增强了控制层的逻辑表达能力和计算效率；然而，大多数公式要么假设标称不确定性，要么依赖于固定噪声模型，当扰动或感知统计偏移时，这会降低满足度[6]。

带有机会约束的随机MPC提供了明确的风险核算，近期分布鲁棒（Distributionally Robust，DR）变体用有限数据证明了样本外安全性。数据使能的DR预测控制提供了即使在扰动分布误设时也保持违规概率低于规定水平的重新公式化[7]。DR机会约束的有限样本集中界限使这些保证定量化和可调[8]。风险厌恶和机会约束MPC的统一视角阐明了何时可处理凸代理保持基本安全属性[9]。输出反馈DR-MPC将这些保证扩展到部分观测情况，并通过循环内估计器设计演示递归可行性[10]。对于机器人操作器，DR紧缩已专门化到扭矩和状态约束，以应对未建模接触和负载变化[11]。与机会约束互补，障碍证书方法推进了安全过滤。随机控制障碍函数（Control Barrier Function，CBF）为扩散驱动系统提供概率一或高置信度界限的前向不变性[12]，近期扩展将构造适应漂移和扩散项中不确定性的安全关键控制器综合[13]。将CBF过滤与逻辑进展联系起来是机器人安全性的新兴主题，特别是当证书必须保留下游规划器和控制器的可行性时[14]。虽然每个方向都产生安全性证据，但在单个MPC循环中整合DR机会约束与逻辑鲁棒性预算还不是标准做法，可能导致过于保守的紧缩或滚动时域上的满足漂移。

时序逻辑规划已超越确定性图搜索，转向明确推理置信度和信息获取的不确定性感知策略。具有拓扑抽象的置信空间规划通过推理同调类同时保持信息论目标来扩展主动SLAM和探索[15]。在移动机器人中，置信空间逻辑规划展示了如何协调规范和部分信息以生成具有实时性能的鲁棒策略[16]。时序逻辑的无遗憾规划开始通过相对于知道潜在结构的预言机的性能来量化部分已知环境中的学习效率[17]。将信息收集与目标达成交织的任务规划在感知和模型不确定性占主导时显示出改善的满足概率[18]。在控制-学习接口，逻辑引导的安全关键学习架构演示了时序规范可以塑造探索和价值估计，同时保持可验证的安全性[19]，并且带有时序逻辑约束的深度学习流程报告了连续系统的理论支持合规[20]。这些方法阐明了规划和学习如何利用逻辑和不确定性；然而，大多数抽象层保证不传播到实现计划的连续动力学，留下了任务级遗憾与动态可行、约束满足执行之间的差距。

前面的文献突出了碎片化：逻辑感知MPC确保在线满足但通常缺乏明确的性能保证；DR机会MPC证明风险但很少推理逻辑鲁棒性预算；置信空间规划和无遗憾策略处理探索和结构学习，但不保证探索参考在执行器限制、概率安全性和估计器动力学下允许可行跟踪。操作的屏蔽机制将逻辑守卫与反馈控制耦合，并演示了对任务扰动的鲁棒性[21]，近期任务-运动管道通过阐明几何和动态约束改善了现实世界的执行保真度[22]。即便如此，一个统一框架能够 (i) 在置信空间优化逻辑鲁棒性，(ii) 执行校准的DR机会约束，(iii) 将抽象无遗憾性能"下沉"到连续动态可行层，以及 (iv) 在滚动时域中证明递归可行性、稳定性和概率满足仍然是一个开放目标。

本研究开发了安全遗憾MPC，一个通过耦合三个设计元素解决上述差距的统一架构。首先，STL目标通过可微鲁棒性代理提升到置信空间，该代理保留经典监控器的排序属性，使得能够在MPC内进行基于梯度的优化。代理与滚动鲁棒性预算配对，该预算跨时域分配时间松弛，使得在分布偏移或估计器瞬态下满足不会侵蚀；该机制直接针对STL的时域耦合性质[1]。其次，通过分布鲁棒机会约束执行安全性，其紧缩是数据驱动的：模糊集在线从残差更新，使得能够在不假设参数噪声模型的情况下控制有限样本违规[7]。当估计器动力学和丰富接触行为占主导时，可以使用输出反馈和非线性变体[10]。第三，通过将规划器的输出视为具有不确定性管道的参考轨迹并证明遗憾转移结果来关闭抽象差距：如果抽象层实现 $o(T)$ 遗憾并且参考遵守速度/曲率和管道可容许条件，则管道/跟踪DR-STL-MPC律的闭环动态遗憾也是 $o(T)$。耦合依赖于分布鲁棒控制不变终端集以及适应机会约束的复合Lyapunov-Barrier参数[12,13]。

提出的框架针对协作装配、移动操作和协作运输，其中任务最自然地由逻辑指定，感知是间歇的或非高斯的，并且不确定性随场景变化而演变。对于此类场景，仅可行性/稳定性证书不能量化相对于安全可行基线的性能，纯学习方法可能在探索期间违反硬约束。通过将置信空间STL、DR机会安全和遗憾分析编织成单个滚动时域控制律，安全遗憾MPC提供具有校准风险和端到端性能保证的可解释满足。该方法通过提供将抽象性能界限转换到最终执行计划的连续、执行器受限层的定量途径来补充屏蔽和任务-运动管道[21,22]。

### 贡献

本文做出四项贡献。首先，本文形式化了一个最优控制问题，该问题最大化置信空间STL鲁棒性，同时执行分布鲁棒机会约束并相对于安全可行策略类界定性能。其次，本文设计了一个可微置信空间STL代理以及滚动鲁棒性预算和针对DR机会约束的在线、残差校准紧缩。第三，本文建立了遗憾转移分析，将抽象无遗憾规划下沉到具有机会约束的动态可行管道/跟踪MPC，在有界跟踪误差下产生 $o(T)$ 动态遗憾。第四，本文通过分布鲁棒控制不变终端集和复合Lyapunov-Barrier函数提供滚动时域证明，获得递归可行性、类ISS稳定性和闭环中满足概率下界。在协作装配和负载运输上的实证研究评估了具有置信界限的满足概率、遗憾曲线、终端和障碍统计以及实时执行；消融量化了每个模块的贡献。

## 2. 预备知识

本节固定符号并回顾整篇论文中使用的基本成分：随机动力学和部分观测、置信状态、具有鲁棒性的信号时序逻辑（STL）以及机会约束（包括分布鲁棒变体）。陈述故意紧凑且自包含，以便新手阅读。

### 2.1 动力学、观测和约束

本研究考虑具有控制输入 $u_k\in\mathcal{U}\subset\mathbb{R}^{m}$、状态 $x_k\in\mathcal{X}\subset\mathbb{R}^{n}$ 和过程扰动 $w_k$ 的离散时间动力学：

```
x_{k+1} = f(x_k,u_k,w_k),    k=0,1,2,...    (1)
```

传感器返回 $y_k=g(x_k,\nu_k)$，其中测量噪声为 $\nu_k$。集合 $\mathcal{X}$ 编码状态约束（例如关节限制），$\mathcal{U}$ 编码执行器约束（例如扭矩/速度界限）。除非另有说明，本文假设 $f$ 和 $g$ 在其状态和输入参数上是局部Lipschitz的。

**安全集和输出映射。** 安全需求由连续可微函数 $h:\mathcal{X}\rightarrow\mathbb{R}$ 编码，其中安全集为：

```
C = {x∈X: h(x) ≥ 0}    (2)
```

多个安全通道 $\{h_j\}_{j=1}^{J}$ 可以聚合为 $h(x) = \min_j h_j(x)$（或通过平滑下逼近）。此外，本文承认任务相关的输出映射 $z = \psi(x)$，这将由STL谓词使用。

### 2.2 信息状态（置信度）和估计器

令 $\mathcal{I}_k:=\{y_{0:k},u_{0:k-1}\}$ 为时间 $k$ 处可用的数据。置信状态 $\beta_k$ 是 $\mathcal{X}$ 上的概率测度，表示给定 $\mathcal{I}_k$ 的 $x_k$ 条件分布。实际上，$\beta_k$ 由滤波器（例如EKF/UKF/粒子滤波器）维护，该滤波器实现：

```
β_{k+1} = BayesUpdate(β_k, u_k, y_{k+1})    (3)
```

并向控制器提供样本或矩。本文用 $\mathbb{E}_{\beta}[\cdot]$ 表示关于 $\beta$ 的期望，并从置信度采样时写作 $x\sim\beta$。

### 2.3 信号时序逻辑（STL）和鲁棒性

实值信号上的STL公式由语法生成：

```
φ ::= ⊤ | μ | ¬φ | φ₁∧φ₂ | G_I φ | F_I φ | φ₁ U_I φ₂
```

其中 $\mu$ 是形式为 $a^\top z \le b$（或更一般地 $p(z)\ge 0$）的原子谓词，$I\subset\mathbb{R}_{\ge 0}$ 是有界时间区间。给定轨迹 $\xi := (x_k)_{k\ge 0}$（或输出 $z_k=\psi(x_k)$），鲁棒性 $\rho(\varphi;\xi,k)$ 是实值映射，其符号指示满足，其幅度量化裕度。本文使用标准语义（布尔连接符的min/max，时间区间上的inf/sup）。为了在MPC内计算，本文采用保持顺序的平滑代理 $\rho^{\text{soft}}$（例如min/max的log-sum-exp近似）；见注记1。

**注记1（平滑鲁棒性）**
令 $\mathrm{smax}_\tau(z):=\tau\log\!\sum_i e^{z_i/\tau}$ 和 $\mathrm{smin}_\tau(z):=-\mathrm{smax}_\tau(-z)$，温度 $\tau>0$。用 $\mathrm{smin}_\tau/\mathrm{smax}_\tau$ 替换min/max产生可微鲁棒性 $\rho^{\text{soft}}$，其相对于精确 $\rho$ 的误差可以由 $\mathcal{O}(\tau\log N)$ 界定，其中 $N$ 是聚合算子中的项数。较小的 $\tau$ 增加保真度但以梯度更陡为代价。

### 2.4 机会约束和分布鲁棒性

固定目标风险水平 $\delta\in(0,1)$。（分阶段）机会约束要求：

```
Pr(h(x_k) ≥ 0) ≥ 1-δ,    对所有 k    (4)
```

由于扰动/测量分布可能被误设并且只有有限数据可用，本文采用分布鲁棒（DR）变体。令 $\widehat{\mathbb{P}}$ 为从最近残差构造的经验分布，$\mathcal{P}$ 为以 $\widehat{\mathbb{P}}$ 为中心的模糊集（例如Wasserstein球或 $\phi$-散度邻域）。DR机会约束要求：

```
inf_{P∈P} Pr_P(h(x_k) ≥ 0) ≥ 1-δ    (5)
```

本文将通过在线校准的可处理紧缩来执行它。

**符号约定。** 本节将时域-$N$ 序列堆叠为 $x_{k:k+N} := (x_k,\dots,x_{k+N})$ 和 $u_{k:k+N-1} := (u_k,\dots,u_{k+N-1})$。指示器 $\mathbf{1}\{\cdot\}$ 当其参数为真时等于1，否则为0。范数 $\|\cdot\|$ 是欧几里得范数，除非另有说明。

## 3. 问题表述

本节现在形式化控制目标、约束和性能标准。问题设置在滚动时域中：在时间 $k$，给定置信度 $\beta_k$ 和当前任务规范，计算输入序列 $u_{k:k+N-1}$，应用 $u_k$，更新 $\beta_{k+1}$，并重复。

### 3.1 任务和安全规范

令 $\varphi$ 为从 $z=\psi(x)$ 上的原子谓词构建的STL任务。本文使用时域窗口上的（软）鲁棒性来量化任务裕度：

```
ρ^{soft}(φ; x_{k:k+N}) ∈ ℝ,
ρ^{soft} > 0 ⇒ 预测满足    (6)
```

安全性由 (2) 中的集合 $\mathcal{C}$ 编码。由于不确定性，需要在时域上的DR机会版本：

```
inf_{P∈P_k} Pr_P(h(x_t) ≥ 0, ∀ t∈{k,…,k+N}) ≥ 1-δ    (7)
```

其中 $\mathcal{P}_k$ 是在时间 $k$ 从最近残差构造的模糊集。本文将 (7) 转换为分阶段紧缩 $h(x_t)\ge \sigma_{k,t}$，其中数据驱动裕度 $\sigma_{k,t}$ 被选择以使联合概率界限成立（第三节提供校准机制）。

### 3.2 置信空间目标和鲁棒性预算

令 $\ell(x,u)$ 为捕获控制努力和标称跟踪目标的阶段成本。在相对于 $\beta_k$ 的期望中控制努力的同时最大化STL鲁棒性：

```
J_k = E_{x∼β_k}[∑_{t=k}^{k+N-1} ℓ(x_t,u_t)] - λ ρ^{soft}(φ; x_{k:k+N})    (8)
```

其中权衡 $\lambda>0$。为了防止由于时域滑动导致的满足侵蚀，本节维护标量鲁棒性预算 $R_k\ge 0$，该预算在应用 $u_k$ 后必须保持非负：

```
R_{k+1} ← UpdateBudget(R_k, ρ^{soft}(φ; x_{k:k+N}))    (9)
```

一个简单的选择是 $R_{k+1}=R_k+\rho^{soft}(\cdot)-\underline{r}$，其中基线 $\underline{r}>0$，但任何证明 $R_k\ge 0$ 的单调更新对于定理1都足够。

### 3.3 策略、安全可行类和遗憾

（非预测）控制策略将信息映射到输入：$\pi:\mathcal{I}_k\mapsto u_k$。本文定义安全可行策略类 $\Pi_{\text{safe}}$：

```
{ π | inf_{P∈P} Pr_P(h(x_t) ≥ 0, ∀ t) ≥ 1-δ, x_t∈X, u_t∈U }    (10)
```

对于有限时域 $T$，定义累积成本 $J_T(\pi):=\mathbb{E}\big[\sum_{k=0}^{T-1}\big(\ell(x_k,u_k)+\lambda\,[ -\rho(\varphi;\xi,k)]_{+}\big)\big]$，其中 $[\cdot]_+=\max\{0,\cdot\}$。策略 $\pi$ 相对于 $\Pi_{\text{safe}}$ 的安全遗憾为：

```
R_T^{safe}(π) := J_T(π) - inf_{π*∈Π_{safe}} J_T(π*)    (11)
```

此外，为了反映连续控制层的动态可行性，本文使用相对于可容许、时变比较器序列 $\{(x_k^\star,u_k^\star)\}$ 的动态遗憾，该序列遵守动力学和约束：

```
R_T^{dyn}(π) := ∑_{k=0}^{T-1}(ℓ(x_k,u_k)-ℓ(x_k^\star,u_k^\star)) + γ∑_{k=0}^{T-1} 1{(x_k,u_k)∉X×U}    (12)
```

其中 $\gamma>0$ 惩罚不可行性。本课题的目标是设计一个滚动时域控制器，在证明安全性和满足的同时保持两个遗憾 $o(T)$。

### 3.4 基本假设

本节收集分析使用的温和正则性和建模假设。

**假设1（正则性）**
映射 $f$ 和 $g$ 在 $(x,u)$ 上是局部Lipschitz的；定义STL原子和 $h(\cdot)$ 的谓词函数是局部Lipschitz的；阶段成本 $\ell$ 在 $u$ 中是凸的，在 $(x,u)$ 上是局部Lipschitz的。

**假设2（估计器和模糊集）**
产生 $\beta_k$ 的滤波器是一致的，其意义是控制器使用的矩或样本是无偏的。模糊集 $\mathcal{P}_k$ 从滑动窗口残差构造并满足有限样本覆盖性质：对于目标 $\delta\in(0,1)$，存在 $\bar{\delta}\in(0,1)$ 使得 $\inf_{\mathbb{P}\in\mathcal{P}_k}\Pr_{\mathbb{P}}(\cdot)\ge 1-\delta$，只要经验违规率 $\le \bar{\delta}$。

**假设3（可行热启动）**
在 $k=0$，存在满足紧缩约束且 $R_0\ge 0$ 的控制序列和终端状态。滚动时域律使用前一个解的热启动。

### 3.5 问题1（具有证明目标的置信空间DR-STL-MPC）

**给定**
- 离散时间随机动力学和部分观测：$x_{t+1}=f(x_t,u_t,w_t)$，$y_t=g(x_t,\nu_t)$，$t\in\mathbb{Z}_{\ge 0}$，具有可容许集 $x_t\in\mathcal{X}\subset\mathbb{R}^n$，$u_t\in\mathcal{U}\subset\mathbb{R}^m$
- 从滤波 $\mathcal{I}_k=\{y_{0:k},u_{0:k-1}\}$ 对 $x_k$ 的置信度 $\beta_k$
- $z=\psi(x)$ 上的STL任务 $\varphi$，具有可微鲁棒性代理 $\rho^{\mathrm{soft}}(\varphi;x_{k:k+N})$
- 安全集 $\mathcal{C}=\{x:\,h(x)\ge 0\}$ 和目标风险水平 $\delta\in(0,1)$
- 用于DR机会约束的模糊集 $\mathcal{P}_k$（从最近残差构建）
- 时域 $N\in\mathbb{Z}_{>0}$，阶段成本 $\ell:\mathcal{X}\times\mathcal{U}\!\to\!\mathbb{R}_{\ge 0}$，终端集 $\mathcal{X}_f$，以及具有更新规则 $\mathsf{UpdateBudget}(\cdot)$ 的鲁棒性预算 $R_k\ge 0$

**决策变量**
预测序列 $(x_{k:k+N},u_{k:k+N-1})$ 和应用的控制 $u_k$

**在时间 $k$ 的优化**

```
min_{x_{k:k+N}, u_{k:k+N-1}}   E_{x∼β_k}[∑_{t=k}^{k+N-1}ℓ(x_t,u_t)] - λ ρ^{soft}(φ;x_{k:k+N})

s.t.
x_{t+1}=f(x_t,u_t,w_t), t=k,…,k+N-1,
x_t∈X, u_t∈U, t=k,…,k+N-1,
h(x_t) ≥ σ_{k,t}, t=k,…,k+N,
x_{k+N}∈X_f,
R_{k+1}=UpdateBudget(R_k,ρ^{soft}(φ;x_{k:k+N})),
R_{k+1} ≥ 0
```

其中紧缩 $\{\sigma_{k,t}\}$ 被校准以使联合DR机会要求成立：

```
inf_{P∈P_k} Pr_P(h(x_t) ≥ 0, ∀ t∈{k,…,k+N}) ≥ 1-δ
```

应用第一个输入 $u_k^\star$，从新数据更新 $(\beta_{k+1},\mathcal{P}_{k+1})$，并在 $k+1$ 重新求解。

**证明目标**
1. **递归可行性**：如果程序在 $k=0$ 可行，则对所有 $k\ge 0$ 保持可行
2. **类ISS稳定性**：存在 $V$ 和类-$\mathcal{K}$ 函数 $\alpha,\sigma$ 使得对所有 $k$，$\mathbb{E}[V(x_{k+1})-V(x_k)] \le -\alpha(\|x_k\|)+\sigma(\|w_k\|)$
3. **概率任务满足**：随着样本增长校准误差 $\varepsilon_n\!\to\!0$，$\inf_{\mathbb{P}\in\mathcal{P}}\Pr_{\mathbb{P}}(\text{$\varphi$ 被满足})\ge 1-\delta-\varepsilon_n$
4. **性能保证**：安全遗憾 $R_T^{\mathrm{safe}}=J_T(\pi)-\inf_{\pi^\star\in\Pi_{\mathrm{safe}}}J_T(\pi^\star)$ 和动态遗憾 $R_T^{\mathrm{dyn}}=\sum_{k=0}^{T-1}(\ell(x_k,u_k)-\ell(x_k^\star,u_k^\star))+\gamma\sum_{k=0}^{T-1}\mathbf{1}\{(x_k,u_k)\notin\mathcal{X}\times\mathcal{U}\}$ 满足当 $T\to\infty$ 时 $R_T^{\mathrm{safe}}=o(T)$ 和 $R_T^{\mathrm{dyn}}=o(T)$

本文的目标是综合一个构造性滚动时域控制器（紧缩校准、预算更新、终端集、管道/跟踪方案），解决优化并满足证明目标。

## 4. 提出的方法

本节开发了问题1的构造性解决方案。从建模选择到可处理约束，然后到连接抽象规划与动态可行控制的管道/跟踪实现。符号在局部引入并在首次使用时总结。

### 4.1 概述和符号

在时间 $k$，控制器保持状态 $x_k\in\mathcal{X}\subset\mathbb{R}^n$ 上的置信度 $\beta_k$，该置信度通过一致滤波器从信息集 $\mathcal{I}_k=\{y_{0:k},u_{0:k-1}\}$ 构造。控制决策是序列 $u_{k:k+N-1}$；只应用 $u_k$，然后时域滑动。目标权衡标称性能 $\ell(x_t,u_t)\ge 0$ 和STL鲁棒性，同时约束执行动力学、执行器界限和具有目标风险 $\delta$ 的分布鲁棒（DR）机会安全性。

本节构建四个组件：
1. 具有滚动鲁棒性预算的可微置信空间STL鲁棒性代理
2. 从残差在线校准的DR机会约束紧缩
3. 分离参考规划与有界跟踪误差的管道/跟踪分解
4. 保证递归可行性的终端集和热启动方案

最终控制器总结在算法1中。

### 4.2 置信空间STL鲁棒性和预算

令 $\varphi$ 为输出 $z_t=\psi(x_t)\in\mathbb{R}^p$ 上的STL公式。精确鲁棒性 $\rho(\varphi; x_{k:k+N})$ 保持顺序但由于min/max和区间inf/sup算子而不可微。本节采用通过将 $\min$ 和 $\max$ 替换为以下内容构建的平滑代理 $\rho^{\text{soft}}$：

```
smin_τ(z₁,…,z_q) := -τ log ∑_{i=1}^q e^{-z_i/τ},
smax_τ(z₁,…,z_q) :=  τ log ∑_{i=1}^q e^{z_i/τ},    (13)
```

其中温度 $\tau>0$。代理保持顺序并承认有界近似误差 $|\rho^{\text{soft}}-\rho|\le \tau \log C(\varphi)$，其中 $C(\varphi)$ 是 $\varphi$ 的任何时间/布尔算子聚合的最大项数。本节通过蒙特卡洛或基于矩的近似在置信空间评估鲁棒性：

```
ρ̃_k := E_{x∼β_k}[ρ^{soft}(φ; x_{k:k+N})]    (14)
```

其中期望使用来自 $\beta_k$ 的样本（粒子集）或无迹点。公式 (14) 以权重 $\lambda>0$ 进入MPC目标。

为了避免时域侵蚀（当时域滑动时满足裕度的逐渐损失），本节跟踪标量鲁棒性预算 $R_k\ge 0$。预算作为时间松弛的简单库存：

```
R_{k+1} = max{0, R_k + ρ̃_k - r̄},    r̄ > 0    (15)
```

本文在MPC约束中要求 $R_{k+1}\ge 0$。常数 $\underline{r}$ 是每步用户选择的基线裕度；如果 $\widetilde{\rho}_k$ 持续低于 $\underline{r}$，$R_k$ 将耗尽，优化器必须通过向近期谓词分配更多松弛来补偿。

### 4.3 通过数据驱动紧缩的分布鲁棒机会约束

安全性由具有安全集 $\mathcal{C}=\{x:\,h(x)\ge 0\}$ 的连续可微函数 $h:\mathcal{X}\to\mathbb{R}$ 指定。本文在时域上执行DR机会要求：

```
inf_{P∈P_k} Pr_P(h(x_t) ≥ 0, ∀ t∈{k,…,k+N}) ≥ 1-δ    (16)
```

直接优化 (16) 很困难。本节将其转换为分阶段紧缩：

```
h(x_t) ≥ σ_{k,t},    t=k,…,k+N,    (17)
```

其中裕度 $\sigma_{k,t}\ge 0$ 被选择以使 (16) 成立。分两步完成：

**步骤1：模糊校准。** 从滑动窗口 $\mathcal{W}_k:=\{r_{k-M+1},\dots,r_k\}$ 上的滤波器残差，本文构造经验分布 $\widehat{\mathbb{P}}_k$ 和模糊集 $\mathcal{P}_k$（例如具有半径 $\epsilon_k$ 的Wasserstein球 $\mathsf{B}_{\epsilon_k}(\widehat{\mathbb{P}}_k)$）。半径 $\epsilon_k$ 被选择以满足有限样本覆盖保证（例如通过集中界限或经验Wasserstein距离的分位数），确保真实扰动分布以高概率位于 $\mathcal{P}_k$ 中。这将样本外违规率控制在期望水平。

**步骤2：确定性裕度。** 在标称预测状态 $\bar{x}_t$ 周围线性化安全函数：$h(x_t)\approx h(\bar{x}_t)+\nabla h(\bar{x}_t)^\top (x_t-\bar{x}_t)$。当预测状态对扰动代理 $\omega_t$ 是仿射的（例如来自线性化或提升模型），$x_t=\bar{x}_t+G_t\omega_t$，单步约束变为：

```
h( x̄_t) + [∇h( x̄_t)^T G_t]_c_t^T ω_t ≥ σ_{k,t}
```

$c_t^\top\omega_t$ 上的DR机会约束承认可处理保守代理。两个常见选项：

1. **分布鲁棒CVaR**：执行

```
CVaR_{δ}^{P_k}(σ_{k,t}-h( x̄_t)-c_t^T ω_t) ≤ 0    (18)
```

这根据模糊集产生线性或二阶锥不等式。

2. **Chebyshev型界限**：如果只有 $\omega_t$ 的均值和协方差 $(\mu_t,\Sigma_t)$ 可靠，执行

```
h( x̄_t)-σ_{k,t} ≥ c_t^T μ_t + κ_δ ‖Σ_t^{1/2} c_t‖_2    (19)
```

其中 $\kappa_\delta$ 是将尾概率上界为 $\delta$ 的Cantelli/Chebyshev因子。 (18) 或 (19) 都产生可计算裕度 $\sigma_{k,t}$。为了从分阶段提升到联合风险 (16)，本节分配每步风险 $\{\delta_t\}$，使得 $\sum_{t=k}^{k+N}\delta_t\le \delta$（布尔不等式），并计算相应裕度。

**关于非线性的注记。** 对于强非线性 $h$，线性化在每个SQP/SCVX迭代时刷新；模糊和紧缩被重新评估，整个程序仍然适合热启动的凸子问题。

### 4.4 管道/跟踪实现和终端集

为了连接抽象参考和动力学，本节使用标准管道MPC分解：

```
x_t = z_t + e_t,    t=k,…,k+N,    (20)
```

其中 $z_t$ 是标称（参考）状态序列，$e_t$ 是限制在从局部反馈设计的管道 $\mathcal{E}$ 中的有界跟踪误差。输入分裂为 $u_t = v_t + K e_t$，其中 $v_t$ 优化标称轨迹，$K$ 是稳定增益（例如来自线性化上的LQR）。通过标准小增益参数，如果扰动集（或协方差）有界，可以选择 $\mathcal{E}$ 使得对所有 $t$，$e_t\in\mathcal{E}$。然后本节在标称部分上紧缩状态/安全约束：

```
z_t ∈ X⊖E,    h(z_t) ≥ σ_{k,t} + η_E    (21)
```

其中 $\ominus$ 是庞特里亚金差，$\eta_{\mathcal{E}}\ge 0$ 补偿 $e_t$ 对 $h(\cdot)$ 的最坏情况影响（从 $h$ 在 $\mathcal{E}$ 上的局部Lipschitz常数计算）。终端集 $\mathcal{X}_f$ 被选择为具有管道反馈的标称动力学的分布鲁棒控制不变；这确保如果 $z_{k+N}\in\mathcal{X}_f$，则可以在相同构造下维持未来可行性。实际上，$\mathcal{X}_f$ 被计算为具有输入 $v_t$ 和反馈 $K$ 的线性化动力学的鲁棒不变集的内逼近；然后根据紧缩安全性 (21) 检查它。

### 4.5 滚动时域程序和求解器

有了前面的元素，在时间 $k$ 求解的置信空间DR-STL-MPC为：

```
min_{z_{k:k+N}, v_{k:k+N-1}}   E_{x∼β_k}[∑_{t=k}^{k+N-1}ℓ(z_t+e_t, v_t+K e_t)] - λ ρ̃_k    (22)

s.t.
z_{t+1} = f̄(z_t,v_t),    t=k,…,k+N-1,    (23)
z_t ∈ X⊖E,    v_t+K e_t ∈ U,    (24)
h(z_t) ≥ σ_{k,t} + η_E,    t=k,…,k+N,    (25)
z_{k+N} ∈ X_f,    (26)
R_{k+1} = max{0, R_k + ρ̃_k - r̄}    (27)
```

其中 $\bar{f}$ 表示标称（可能线性化）动力学，$e_t$ 是由局部误差模型预测的管道误差（或由设计有界）。 (22) 中的期望通过绘制样本 $x\sim\beta_k$ 并通过 (20) 传播它们来评估；使用二次 $\ell$ 和凸紧缩，每个SQP/SCVX迭代减少到从前一个解热启动的QP/SOCP。

**热启动和实时操作。** 本节存储前一个优化器 $(z^\star,v^\star)$，将其移动一步，并将其用作初始猜测。此外还热启动紧缩约束的对偶变量。模糊半径 $\epsilon_k$、裕度 $\sigma_{k,t}$ 和温度 $\tau$ 缓慢更新以避免振荡。

### 4.6 从抽象无遗憾到动态遗憾（草图）

令规划器（抽象层）产生参考序列 $\{z_t^{\text{ref}}\}$，具有 $o(T)$ 遗憾，相对于其类中的比较器（例如具有完整地图知识的预言机）。假设 (1) $z_t^{\text{ref}}$ 满足与管道 $\mathcal{E}$ 和输入分裂 $u_t=v_t+Ke_t$ 兼容的速度/曲率界限，并且 (2) 模糊校准和紧缩满足4.3节的覆盖性质。那么管道/跟踪MPC (22)-(27) 确保有界跟踪误差 $e_t$ 和可行输入。与动态可行比较器的累积成本差异可以由抽象遗憾加上随 $\sum_t\|e_t\|$ 和紧缩松弛缩放的项界定，产生次线性动态遗憾 $R_T^{\text{dyn}}=o(T)$；安全遗憾通过将参数限制到安全可行类由相同参数控制。

### 4.7 算法

算法1实现了4.1-4.5节描述的控制器，明确说明了每步计算的内容和原因。本节回顾关键符号，因为它们首次出现：$\beta_k$ 是状态上的当前置信度（用于绘制期望样本的分布）；$\mathcal{W}_k$ 是滤波器残差的滑动窗口；$\widehat{\mathbb{P}}_k$ 是从 $\mathcal{W}_k$ 估计的经验扰动律；$\mathcal{P}_k$ 是以 $\widehat{\mathbb{P}}_k$ 为中心的模糊集（例如Wasserstein球）；$\{\delta_t\}$ 是具有 $\sum_{t=k}^{k+N}\delta_t\le\delta$ 的每步风险分配；$\{\sigma_{k,t}\}$ 是使联合机会约束成立的确定性安全裕度（紧缩）；$\widetilde{\rho}_k$ 是置信空间STL鲁棒性代理；$(K,\mathcal{E})$ 定义管道反馈和误差集；$\mathcal{X}_f$ 是鲁棒终端集；$R_k$ 是标量鲁棒性预算。内部程序的优化变量是标称状态/输入 $(z_{k:k+N},v_{k:k+N-1})$；应用的控制是 $u_k^\star=v_k^\star+K e_k^\star$，其中 $e_k^\star$ 是时间 $k$ 处的预测管道误差。

**估计**更新用于成本（通过 $\widetilde{\rho}_k$）和不确定性量化（通过残差）的信息状态。**模糊校准**通过将 $\widehat{\mathbb{P}}_k$ 扩大为 $\mathcal{P}_k$ 将有限数据转换为样本外保证。**紧缩**将分布机会要求转换为易于在优化器中施加的确定性裕度。**鲁棒性评估**计算保持满足顺序并且可以相对于决策变量微分的平滑STL裕度。**求解MPC**执行动力学、界限、管道约束、安全裕度、终端不变性和预算非负性，返回第一个控制。**应用和更新**关闭循环并更新 $R_{k+1}$ 以防止时域侵蚀。

**算法1（安全遗憾MPC：具有管道/跟踪和校准紧缩的置信空间DR-STL）**

**输入（固定）：** STL公式 $\varphi$；时域 $N$；总风险 $\delta$；权重 $\lambda>0$；基线鲁棒性 $\underline{r}>0$；管道反馈 $K$ 和误差集 $\mathcal{E}$；终端集 $\mathcal{X}_f$；软鲁棒性温度 $\tau>0$；残差窗口大小 $M$。

**初始化：** 一致状态估计器（以产生 $\beta_0$）；满足紧缩约束的可行热启动 $(z_{0:N},v_{0:N-1})$；预算 $R_0\ge 0$。

**对于 $k=0,1,2,\dots$：**
1. **估计（置信度和残差）。** 使用选定滤波器从 $\mathcal{I}_k=\{y_{0:k},u_{0:k-1}\}$ 更新置信度 $\beta_k$；将最近的 $M$ 个残差收集到 $\mathcal{W}_k$ 中。
2. **模糊校准（有限样本覆盖）。** 从 $\mathcal{W}_k$ 形成经验扰动律 $\widehat{\mathbb{P}}_k$；从覆盖规则（例如经验Wasserstein距离的分位数或集中界限）选择半径 $\epsilon_k$，并设置模糊集 $\mathcal{P}_k=\mathsf{B}_{\epsilon_k}(\widehat{\mathbb{P}}_k)$。
3. **风险分配和紧缩（确定性裕度）。** 跨时域分配总风险以获得 $\{\delta_t\}_{t=k}^{k+N}$，其中 $\sum \delta_t\le \delta$（均匀或截止时间偏置）；通过对每步执行DR-CVaR（凸）或Chebyshev型界限计算分阶段裕度 $\{\sigma_{k,t}\}$；在映射到标称状态 $z_t$ 时添加管道修正 $\eta_{\mathcal{E}}$。
4. **鲁棒性代理（置信空间评估）。** 从 $\beta_k$ 绘制样本 $\{x^{(s)}\}_{s=1}^{S}$（或使用sigma点）并使用 (13) 评估软鲁棒性 $\rho^{\text{soft}}(\varphi;x^{(s)}_{k:k+N})$；平均以获得 (14) 中的 $\widetilde{\rho}_k$。
5. **求解DR-STL管道MPC（热启动）。** 使用从移动的前一个解获得的初始猜测，对标称问题 (22)-(27) 求解 $(z_{k:k+N}^\star,v_{k:k+N-1}^\star)$，受动力学、界限、管道约束、安全裕度 $h(z_t)\ge \sigma_{k,t}+\eta_{\mathcal{E}}$、终端约束 $z_{k+N}\in\mathcal{X}_f$ 和预算更新 $R_{k+1}=\max\{0,R_k+\widetilde{\rho}_k-\underline{r}\}$ 约束。
6. **应用和更新（滑动时域）。** 构造 $u_k^\star=v_k^\star+K e_k^\star$ 并将其应用于系统；接收 $y_{k+1}$，更新 $\beta_{k+1}$ 和 $\mathcal{W}_{k+1}$，如上设置 $R_{k+1}$，将 $(z^\star,v^\star)$ 移动一步以热启动下一次迭代，并设置 $k\leftarrow k+1$。

## 5. 理论基础

本节证明问题1的控制器。结果组织为每个陈述针对一个证明目标（可行性、稳定性、概率满足和性能）。

### 5.1 基本假设和技术预备知识

**假设4（管道动力学、估计器和终端集）**
1. **管道动力学**：存在稳定增益 $K$ 和紧致误差集 $\mathcal{E}$ 使得跟踪误差 $e_{t+1}=\phi(e_t,\omega_t)$ 是局部Lipschitz的，满足 $\phi(0,0)=0$，并且当 $\omega_t$ 有界时 $e_t\in\mathcal{E}$ 意味着 $e_{t+1}\in\mathcal{E}$。
2. **估计器**：置信度 $\beta_k$ 由一致滤波器产生，因此 (14) 中使用的基于样本的期望是无偏的。
3. **模糊覆盖**：数据驱动模糊集 $\mathcal{P}_k$ 满足有限样本覆盖保证：对于任何事件 $A$，$\inf_{\mathbb{P}\in\mathcal{P}_k}\Pr_{\mathbb{P}}(A)\ge \Pr(A)-\varepsilon_n$，其中当残差窗口 $n\!\to\!\infty$ 时 $\varepsilon_n\!\to\!0$。
4. **终端集**：存在标称动力学 $z_{t+1}=\bar f(z_t,v_t)$ 的分布鲁棒控制不变集 $\mathcal{X}_f\subseteq \mathcal{X}\ominus \mathcal{E}$。

管道不变性将标称优化器 $(z,v)$ 连接到物理状态 $x=z+e$，使得在 $z$ 上的紧缩意味着 $x$ 的可行性。估计器一致性和模糊覆盖证明了在机会约束中用 $\mathcal{P}_k$ 替换未知扰动律的合理性。终端不变性是MPC"关闭时域"的标准设备，此处将用它随时间向前传播可行性。

**引理1（平滑STL代理：均匀误差界限和可微性）**
令 $\rho(\varphi;\xi_{k:k+N})$ 为精确STL鲁棒性（min/max布尔语义；在离散时间中inf/sup时间语义简化为有限min/max）。令 $\rho^{\mathrm{soft}}$ 通过将 $\{z_i\}_{i=1}^{q}$ 上的每个有限min/max替换为以下内容获得：

```
smin_τ(z) := -τ log ∑_{i=1}^{q}e^{-z_i/τ},
smax_τ(z) :=  τ log ∑_{i=1}^{q}e^{z_i/τ},    τ > 0
```

令 $\mathcal{T}(\varphi)$ 为语法树，$q(v)$ 为节点 $v$ 的扇入（聚合项数）。定义聚合复杂度：

```
C(φ) := max_P ∏_{v∈P} q(v)
```

其中最大值是 $\mathcal{T}(\varphi)$ 中所有从根到叶路径 $P$ 上的。那么对于任何段 $\xi_{k:k+N}$，

```
|ρ^{soft}(φ;ξ_{k:k+N}) - ρ(φ;ξ_{k:k+N})| ≤ τ log C(φ)
```

如果每个原子谓词映射 $p_j$ 在其参数中是 $C^1$，则 $\rho^{\mathrm{soft}}$ 在所有谓词值（因此在 $z_t=\psi(x_t)$ 和 $x_t$ 任何 $\psi$ 是 $C^1$ 的地方）中是 $C^1$。

**证明：**
1. **一步近似。** 对于任何有限集 $\{z_i\}_{i=1}^{q}$ 和 $\tau>0$，

```
max_i z_i ≤ smax_τ(z) ≤ max_i z_i + τ log q,
min_i z_i ≥ smin_τ(z) ≥ min_i z_i - τ log q
```

这些从 $\log\!\sum_i e^{z_i/\tau}\le \max_i(z_i/\tau)+\log q$ 及其对称形式得出。

2. **树形累积。** 在语法树 $\mathcal{T}(\varphi)$ 上自底向上评估鲁棒性：每个内部节点 $v$ 对其 $q(v)$ 个子节点应用min或max（离散时间中的时间算子简化为有限min/max）。用 $\mathrm{smin}_\tau/\mathrm{smax}_\tau$ 替换每个此类聚合以获得 $\rho^{\mathrm{soft}}$。通过结构归纳进行。在叶节点，不进行近似（误差0）。在具有子节点误差 $\{E_i\}$ 的内部节点 $v$ 处，$\mathrm{smin}_\tau/\mathrm{smax}_\tau$ 关于 $\ell_\infty$ 扰动的1-Lipschitz性质给出 $|\mathrm{s(\cdot)}(x^{\mathrm{soft}})-\mathrm{s(\cdot)}(x)|\le \max_i E_i$，应用 中的界限最多增加 $\tau\log q(v)$。因此 $v$ 处的误差界为 $\tau\log q(v)+\max_i E_i$，即沿一条从根到 $v$ 的路径求和 $\tau\log q(\cdot)$。在根节点，这产生陈述的界限 $\tau\log C(\varphi)$。

3. **可微性。** $\mathrm{smin}_\tau,\mathrm{smax}_\tau$ 是 $C^\infty$，梯度等于softmin/softmax权重。将有限多个 $C^\infty$ 聚合与 $C^1$ 谓词复合通过链规则保持 $C^1$。

(22)-(27) 中的优化器需要梯度。引理1证明用 $\rho^{\mathrm{soft}}$ 替换 $\rho$（以获得导数）引入与 $\tau$ 成比例的受控偏差，并且当裕度大于此偏差时保留鲁棒性的符号。这使算法1能够将满足视为平滑目标/约束，同时保持语义保真度。

### 5.2 从分布鲁棒机会约束到确定性紧缩

**引理2（意味着联合DR机会约束的确定性裕度）**
令 $h:\mathcal{X}\!\to\!\mathbb{R}$ 在 $\mathcal{X}\ominus\mathcal{E}$ 上是 $L_h$-Lipschitz的，并使用管道分解 $x_t=z_t+e_t$，其中 $e_t\in\mathcal{E}$ 且 $\|e_t\|\le\bar e$。对于每个 $t\in\{k,\dots,k{+}N\}$，定义 $c_t:=\nabla h(z_t)$ 和标量扰动代理 $Y_t:=c_t^\top\omega_t$，其中 $\omega_t$ 收集沿敏感度方向 $c_t$ 影响 $x_t$ 的不确定性。假设在任何 $\mathbb{P}\in\mathcal{P}_k$ 下，$Y_t$ 具有均值 $\mu_t:=\mathbb{E}_{\mathbb{P}}[Y_t]$ 和方差 $\sigma_t^2:=\mathrm{Var}_{\mathbb{P}}[Y_t] = c_t^\top\Sigma_t c_t$，其中 $\Sigma_t$ 是 $\omega_t$ 的协方差。如果分阶段裕度 $\sigma_{k,t}$ 满足

```
h(z_t) ≥ L_h ē + μ_t + κ_{δ_t} σ_t + σ_{k,t}    (28)
```

其中 $\kappa_{\delta_t}:=\sqrt{(1-\delta_t)/\delta_t}$（Cantelli/Chebyshev因子）且 $\sum_{t=k}^{k+N}\delta_t\le \delta$，则

```
inf_{P∈P_k} Pr_P(h(x_t) ≥ σ_{k,t}, ∀ t∈{k,…,k+N}) ≥ 1-δ-ε_n
```

其中 $\varepsilon_n\!\to\!0$ 是假设4中的模糊覆盖误差。

**证明：**
1. **分离管道误差和随机变化。** 通过 $h$ 在 $\mathcal{X}\ominus\mathcal{E}$ 上的 $L_h$-Lipschitz连续性和 $x_t=z_t+e_t$，

```
h(x_t) = h(z_t+e_t) ≥ h(z_t) - L_h‖e_t‖ ≥ h(z_t) - L_h ē    (29)
```

$h(z_t)$ 中由于过程/估计噪声的剩余不确定性由代理 $Y_t=c_t^\top\omega_t$ 捕获，该代理表示 $h$ 在 $z_t$ 处沿敏感度方向 $c_t$ 的一阶变化。$h(x_t)\!\ge\!\sigma_{k,t}$ 的充分（保守）条件因此是

```
h(z_t) - L_h ē + Y_t ≥ σ_{k,t}    (30)
```

2. **分布鲁棒一步界限。** 固定 $t$ 并考虑在任何 $\mathbb{P}\in\mathcal{P}_k$ 下具有均值 $\mu_t$ 和方差 $\sigma_t^2$ 的随机标量 $Y_t$。Cantelli（单边Chebyshev）不等式指出对于任何 $a>0$，

```
Pr_P(Y_t - μ_t ≤ -a) ≤ σ_t^2/(σ_t^2 + a^2)
```

选择 $a=\kappa_{\delta_t}\sigma_t$，其中 $\kappa_{\delta_t}:=\sqrt{(1-\delta_t)/\delta_t}$，使得右边等于 $\delta_t$。等价地，

```
Pr_P(Y_t ≥ μ_t - κ_{δ_t}σ_t) ≥ 1-δ_t,    ∀ P∈P_k    (31)
```

结合 (30) 和 (31)，分阶段DR机会约束 $\inf_{\mathbb{P}\in\mathcal{P}_k}\Pr_{\mathbb{P}}(h(x_t)\ge \sigma_{k,t})\ge 1-\delta_t$ 的充分条件是

```
h(z_t) - L_h ē + μ_t - κ_{δ_t}σ_t ≥ σ_{k,t}
```

这重新排列为 (28)。

3. **从分阶段到联合机会约束。** 令 $E_t:=\{h(x_t)\ge \sigma_{k,t}\}$，对于 $t=k,\dots,k{+}N$。通过联合界限，

```
Pr_P(∩_{t=k}^{k+N} E_t) ≥ 1 - ∑_{t=k}^{k+N} Pr_P(E_t^c) ≥ 1 - ∑_{t=k}^{k+N}δ_t ≥ 1-δ
```

并且通过 (28) 的构造，该界限对所有 $\mathbb{P}\in\mathcal{P}_k$ 均匀成立。

4. **模糊覆盖。** 假设4保证数据驱动模糊集 $\mathcal{P}_k$ 覆盖真实扰动律直到 $\varepsilon_n$，即对于任何事件 $A$，$\inf_{\mathbb{P}\in\mathcal{P}_k}\Pr_{\mathbb{P}}(A)\ge \Pr(A)-\varepsilon_n$。将此应用于 $A=\bigcap_{t=k}^{k+N}E_t$ 产生陈述的联合界限

```
inf_{P∈P_k} Pr_P(h(x_t) ≥ σ_{k,t}, ∀ t) ≥ 1-δ-ε_n
```

5. **保守性总结。** 不等式 (29) 当 $h$ 沿 $e_t$ 以速率 $L_h$ 变化时紧；Cantelli界限是给定 $(\mu_t,\sigma_t^2)$ 时最少信息的分布鲁棒尾界限。任何额外结构（例如次高斯尾或DR-CVaR模糊）可以通过用相应的更紧密风险缓冲区替换 $\kappa_{\delta_t}\sigma_t$ 来结合，而不改变证明结构。

公式 (28) 是概率安全要求和优化器确定性约束之间的桥梁。它确切告诉我们紧缩 $\sigma_{k,t}$ 必须多大，考虑 (1) 管道误差，以及 (2) $\mathcal{P}_k$ 内由 $(\mu_t,\Sigma_t)$ 总结的分布不确定性。这在算法1的"紧缩"块中实现。

**注记（替代DR-CVaR紧缩）**
如果 $\mathcal{P}_k$ 是Wasserstein或 $\phi$-散度球，可以用鲁棒CVaR约束 $\mathrm{CVaR}^{\mathcal{P}_k}_{\delta_t}(\sigma_{k,t}-h(z_t)-c_t^\top\omega_t)\le 0$ 替换Chebyshev项，这导致形式为 $h(z_t)\ge \sigma_{k,t}+\theta_{k,t}$ 的二阶锥约束。算法和下面的证明保持不变；只有裕度的公式不同。

### 5.3 滚动时域循环的可行性和稳定性

**定理1（MPC程序的递归可行性）**
假设假设4成立并且问题 (22)-(27) 在 $k=0$ 可行。那么，通过引理2计算的裕度和鲁棒性预算更新 $R_{k+1}=\max\{0,R_k+\widetilde\rho_k-\underline r\}$，问题对所有 $k\ge 0$ 可行。

**证明：**
通过对 $k$ 归纳进行。

**基本情况（$k=0$）。** 根据假设，存在可行对 $(z_{0:N},v_{0:N-1})$ 满足 (23)-(26)，具有来自引理2的裕度和预算 $R_0\ge 0$。

**归纳步骤。** 假设在时间 $k$ 存在 (22)-(27) 的可行解 $(z_{k:k+N},v_{k:k+N-1})$，具有裕度 $\{\sigma_{k,t}\}$ 和预算 $R_k\ge 0$。为时间 $k{+}1$ 构造移动候选并证明它满足 $k{+}1$ 处问题的所有约束。

定义

```
z̃_{k+1:k+N} := z_{k+1:k+N},    ṽ_{k+1:k+N-1} := v_{k+1:k+N-1}
```

根据假设4（$\mathcal{X}_f$ 的分布鲁棒控制不变性），存在终端控制律 $\kappa:\mathcal{X}_f\to\mathcal{U}$ 使得对于 $z\in\mathcal{X}_f$，$\bar f(z,\kappa(z))\in\mathcal{X}_f$ 并且在终端步骤满足紧缩安全条件。设置

```
z̃_{k+N+1} := f̄(z_{k+N}, κ(z_{k+N})),    ṽ_{k+N} := κ(z_{k+N})
```

这为 $k{+}1$ 处的时域定义了完整的标称序列 $(\tilde z_{k+1:k+N+1},\tilde v_{k+1:k+N})$。

根据构造，在时间 $k{+}1$ 用 $(\tilde z,\tilde v)$ 满足标称动力学 (23)。在时间 $k$，可行性意味着 $z_t\in\mathcal{X}\ominus\mathcal{E}$ 并且 $(v_t+Ke_t)\in\mathcal{U}$，对于 $t=k,\dots,k{+}N{-}1$。因此 $\tilde z_t=z_t\in\mathcal{X}\ominus\mathcal{E}$ 并且 $(\tilde v_t+Ke_t)=(v_t+Ke_t)\in\mathcal{U}$，对于 $t=k{+}1,\dots,k{+}N{-}1$。在新的终端步骤，$\tilde z_{k+N+1}\in\mathcal{X}_f\subseteq\mathcal{X}\ominus\mathcal{E}$ 通过不变性，并且关联的输入 $\tilde v_{k+N}=\kappa(z_{k+N})$ 根据终端控制器的定义是可容许的。最后，假设4 授予管道不变性：如果 $e_t\in\mathcal{E}$ 且 $\|\omega_t\|\le\bar\omega$，则 $e_{t+1}\in\mathcal{E}$。因此只要标称变量满足紧缩约束，如 (24) 中，物理状态 $x_t=\tilde z_t+e_t$ 和输入 $u_t=\tilde v_t+Ke_t$ 就遵守硬界限。

在时间 $k{+}1$，使用引理2用更新的置信度/模糊集和标称序列 $(\tilde z_{k+1:k+N+1},\tilde v_{k+1:k+N})$ 重新计算裕度 $\{\sigma_{k+1,t}\}_{t=k+1}^{k+N+1}$。引理2确保分阶段确定性约束 $h(\tilde z_t)\ge \sigma_{k+1,t}+\eta_{\mathcal{E}}$ 意味着直到有限样本覆盖误差 $\varepsilon_n$ 的时域 $[k{+}1,k{+}N{+}1]$ 上的联合分布鲁棒机会约束。因为 $\tilde z_t=z_t$ 对于 $t=k{+}1,\dots,k{+}N$，时间 $k$ 的可行性与更新裕度一起对这些步骤强制 (25)；在 $t=k{+}N{+}1$，终端不变性通过 $\kappa$ 和 $\mathcal{X}_f$ 的设计保证 $h(\tilde z_{k+N+1})$ 满足紧缩界限。因此 $k{+}1$ 处的整个时域满足紧缩安全约束，并且通过引理2联合DR机会要求成立。

根据构造，$\tilde z_{k+N+1}\in\mathcal{X}_f$，满足终端约束。对于预算，时间 $k$ 的可行性意味着 $R_k\ge 0$ 并且更新规则产生

```
R_{k+1} = max{0, R_k + ρ̃_k - r̄} ≥ 0
```

所以 $k{+}1$ 处的预算约束满足。预算是优化中的额外线性状态并且不影响上述可行性参数。

在时间 $k{+}1$ 产生了可行候选 $(\tilde z_{k+1:k+N+1},\tilde v_{k+1:k+N})$，满足 (23)-(26) 具有重新计算的裕度和更新预算。因此问题 (22)-(27) 在 $k{+}1}$ 可行。通过归纳，可行性对所有 $k\ge 0$ 成立。

**注记**
证明确立了 $k{+}1$ 处可行点的存在。实际上，求解器用移动序列 $(\tilde z,\tilde v)$ 热启动，这确保快速收敛。如果动力学和安全函数通过序列凸近似执行，候选被评估和重新线性化；可行性保留，因为紧缩在每次更新后重新计算并且终端集保持不变。因此，数值实现反映了证明中使用的构造性参数。

定理1确保控制器一旦可行启动就不会由于不可行性而"暂停"。这是使算法在规定风险下无限期运行的主干，这是TRO应用中长时域任务的要求。

### 5.4 STL规范的概率满足

**定理2（满足概率下界）**
令 $\rho$ 为闭环轨迹的精确STL鲁棒性，$\rho^{\mathrm{soft}}$ 为其平滑代理（引理1）。在引理2的紧缩构造下，具有风险分配 $\sum_{t=k}^{k+N}\delta_t\le \delta$ 在每个时间 $k$，以及预算更新 $R_{k+1}=\max\{0,R_k+\widetilde\rho_k-\underline r\}$，算法1生成的闭环对于任何有限时域 $T$ 满足

```
inf_{P∈P} Pr_P(ρ(φ;ξ,k) ≥ 0, ∀ k=0,…,T) ≥ 1-δ-ε_n-η_τ
```

其中 $\eta_\tau=\mathcal{O}(\tau\log C(\varphi))$。当随着数据积累模糊覆盖误差 $\varepsilon_n\to 0$ 且平滑温度 $\tau$ 固定时，相同的界限在极限 $T\to\infty$ 中成立。

**证明：**
1. **窗口式概率安全。** 固定 $k$ 并对确定 $(\beta_k,\mathcal{P}_k)$ 和裕度 $\{\sigma_{k,t}\}_{t=k}^{k+N}$ 的信息集 $\mathcal{I}_k$ 条件。通过引理2和 $\mathcal{P}_k$ 的有限样本覆盖，

```
inf_{P∈P_k} Pr_P(h(x_t) ≥ σ_{k,t}, ∀ t∈{k,…,k+N} | I_k) ≥ 1-δ-ε_n    (32)
```

递归可行性（定理1）确保优化器在每个 $k$ 实际执行紧缩约束。

2. **从安全到预测满足。** 在窗口 $[k,k+N]$ 内，可行性和预算更新意味着 $R_{k+1}=\max\{0,R_k+\widetilde\rho_k-\underline r\}\ge 0$。因此 $\widetilde\rho_k\ge \underline r - R_k$；选择 $\underline r>0$ 并初始化 $R_0\ge 0$ 产生当 $R_k$ 不耗尽时 $\widetilde\rho_k\ge 0$。由于 $R_k$ 在可用时累积正裕度，标准望远镜求和显示 $\frac{1}{K}\sum_{k=0}^{K-1}\widetilde\rho_k \ge \underline r - \frac{R_K}{K}$，因此对于有限 $K$，存在 $\widetilde\rho_k\ge 0$ 的索引子集。与 (32) 结合，可得在至少 $1-\delta-\varepsilon_n$ 概率的事件上，$[k,k+N]$ 上的预测（软）鲁棒性是非负的。

3. **软到精确鲁棒性。** 通过引理1，软鲁棒性用均匀界限近似精确鲁棒性：对于任何长度 $N$ 的轨迹段，

```
|ρ^{soft}(φ;ξ_{k:k+N}) - ρ(φ;ξ_{k:k+N})| ≤ η_τ,    η_τ = O(τ log C(φ))
```

因此如果 $\rho^{\mathrm{soft}}(\varphi;\xi_{k:k+N})\ge 0$，则 $\rho(\varphi;\xi_{k:k+N})\ge -\eta_\tau$；特别地，除了在最坏情况下概率上界为 $\eta_\tau$ 的事件外，符号一致。

4. **塔属性和时间的联合界限。** 令 $E_k$ 为事件"精确鲁棒性在 $[k,k+N]$ 上满足STL规范。"从步骤1-3，$\Pr_{\mathbb{P}}(E_k\,|\,\mathcal{I}_k)\ge 1-\delta-\varepsilon_n-\eta_\tau$ 对 $\mathbb{P}\in\mathcal{P}$ 均匀。通过塔属性，$\Pr_{\mathbb{P}}(E_k)\ge 1-\delta-\varepsilon_n-\eta_\tau$。对于任何有限 $T$，事件"$\rho(\varphi;\xi,k)\ge 0$ 对所有 $k\le T$"包含在 $\bigcap_{j\in\mathcal{K}_T} E_j$ 中，其中 $\mathcal{K}_T$ 是任何其窗口覆盖 $[0,T]$ 的索引集（例如 $\mathcal{K}_T=\{0,N{+}1,2(N{+}1),\dots\}$）。应用布尔不等式，其中 $|\mathcal{K}_T|\le \lceil (T{+}1)/(N{+}1)\rceil$，以及均匀每窗口下界产生

```
Pr_P(ρ(φ;ξ,k) ≥ 0, ∀ k ≤ T) ≥ 1 - (δ + ε_n + η_τ)
```

注意窗口的重叠防止界限随 $T$ 恶化（每窗口保证已经涉及整个窗口）。对 $\mathbb{P}\in\mathcal{P}$ 取下确界保留不等式。

5. **极限 $T\to\infty$。** 如果模糊覆盖误差随数据减少（$\varepsilon_n\to 0$ 沿运行）且 $\tau$ 固定，相同界限在 $T$ 中均匀成立。

**注记**
常数 $\delta$（风险）、$\varepsilon_n$（有限样本覆盖）和 $\eta_\tau$（平滑偏差）分别考虑分布不确定性、数据稀缺和代理近似。减少其中任何一个收紧界限而不改变证明。

定理2将算法的两个关键近似——分布鲁棒性和软鲁棒性——连接到全局保证：任务在整个执行中以高概率满足，直到明确的校准和平滑误差。

### 5.5 性能：从抽象无遗憾到动态和安全遗憾

**引理3（跟踪误差累积界限）**
令管道误差在局部模型下演化

```
e_{k+1} = (A+BK)e_k + Gω_k + Δ_k    (33)
```

其中 $(A{+}BK)$ 是Schur的，$\omega_k$ 聚合过程/估计噪声。假设：
1. $\{\omega_k\}$ 具有有界二阶矩，$\sup_k \mathbb{E}\|\omega_k\|^2 \le \bar\sigma_\omega^2$
2. 线性化残差 $\Delta_k$ 满足 $\mathbb{E}\|\Delta_k\|^2 \le \bar\sigma_\Delta^2$ 并且与 $e_k$ 不相关
3. 假设4的管道不变性成立，使得 (33) 的解保持在 (33) 有效的区域中

那么存在 $C_e>0$ 使得对于所有 $T\ge 1$，

```
∑_{k=0}^{T-1} E‖e_k‖ ≤ C_e T^{1/2}
```

此外，如果误差动力学在以下意义下是均方稳定的：存在 $\beta\!\in\!(0,1)$，$M\!<\!\infty$ 使得对于所有 $k$，$\mathbb{E}\|e_k\|^2 \le M\beta^k \|e_0\|^2 + M(\bar\sigma_\omega^2+\bar\sigma_\Delta^2)$，则

```
∑_{k=0}^{T-1} E‖e_k‖^2 ≤ C_e
```

**证明：**
由于 $(A{+}BK)$ 是Schur的，存在 $P\succ 0$ 和 $Q\succ 0$ 求解离散Lyapunov方程

```
(A+BK)^T P (A+BK) - P = -Q
```

定义 $V_k:=e_k^\top P e_k$ 和 $Q_{e_k} = (A+BK)e_k+G\omega_k+\Delta_k$。从 (33) 和展开，

```
E[V_{k+1}|F_k] = E[Q_{e_k}^T P Q_{e_k}|F_k]
            = e_k^T (A+BK)^T P (A+BK) e_k
              + 2 E[e_k^T (A+BK)^T P (Gω_k+Δ_k)|F_k]
              + E[(Gω_k+Δ_k)^T P (Gω_k+Δ_k)]
```

在所述的无相关性和扰动零均值（或通过完全平方并使用Young不等式界定混合项）下，交叉项消失（或通过调整 $Q$ 吸收到 $\|e_k\|^2+\|\omega_k\|^2+\|\Delta_k\|^2$ 的倍数中）。使用Lyapunov恒等式，

```
e_k^T (A+BK)^T P (A+BK) e_k = V_k - e_k^T Q e_k ≤ V_k - λ_min(Q)‖e_k‖^2
```

对于噪声项，$E[(Gω_k+Δ_k)^T P (Gω_k+Δ_k)] ≤ λ_max(P) E‖Gω_k+Δ_k‖^2 ≤ 2λ_max(P)(‖G‖^2 σ̄_ω^2 + σ̄_Δ^2)$。取总期望产生

```
EV_{k+1} ≤ EV_k - λ_min(Q) E‖e_k‖^2 + c_0,    c_0 := 2λ_max(P)(‖G‖^2σ̄_ω^2 + σ̄_Δ^2)    (34)
```

求和 (34) 对于 $k=0,\dots,T-1$ 给出

```
λ_min(Q)∑_{k=0}^{T-1} E‖e_k‖^2 ≤ EV_0 - EV_T + c_0 T ≤ e_0^T P e_0 + c_0 T
```

因此

```
∑_{k=0}^{T-1} E‖e_k‖^2 ≤ (e_0^T P e_0)/λ_min(Q) + (c_0/λ_min(Q)) T    (35)
```

通过Cauchy-Schwarz，

```
∑_{k=0}^{T-1} E‖e_k‖ ≤ √T (∑_{k=0}^{T-1} E‖e_k‖^2)^{1/2} ≤ √T ((e_0^T P e_0)/λ_min(Q) + (c_0/λ_min(Q)) T)^{1/2}
```

吸收常数到 $C_e$ 产生 $\sum_{k=0}^{T-1}\mathbb{E}\|e_k\|$ 上的陈述 $\mathcal{O}(\sqrt{T})$ 界限。

**步骤4：均方稳定情况。** 如果 $\mathbb{E}\|e_k\|^2 \le M\beta^k \|e_0\|^2 + M(\bar\sigma_\omega^2+\bar\sigma_\Delta^2)$，则

```
∑_{k=0}^{T-1} E‖e_k‖^2 ≤ M‖e_0‖^2 ∑_{k=0}^{∞} β^k + M(σ̄_ω^2+σ̄_Δ^2) T
```

此外，因为管道不变性将 $e_k$ 限制在紧致 $\mathcal{E}$，稳态二阶矩独立于 $T$ 有界。结合这些事实给出统一界限 $\sum_{k=0}^{T-1} \mathbb{E}\|e_k\|^2 \le C_e$，其中 $C_e$ 取决于 $(M,\beta,\|e_0\|,\bar\sigma_\omega,\bar\sigma_\Delta)$ 和管道常数。

**注记**
$C_e$ 中的常数取决于Lyapunov对 $(P,Q)$、扰动协方差和管道设计 $(K,\mathcal{E})$。更紧的管道（更小的 $\bar e$）减少线性化残差 $\Delta_k$ 并因此改善 $C_e$。

当将遗憾从抽象规划器下沉到物理控制器时，主要损失来自跟踪误差。引理3确保损失增长慢于 $T$，这是在下一个定理中建立次线性动态遗憾的技术杠杆。

**定理3（遗憾转移：抽象 → 动态和安全）**
令抽象参考生成器产生标称序列 $\{z_k^{\mathrm{ref}},v_k^{\mathrm{ref}}\}_{k=0}^{T-1}$，该序列对管道是动态可容许的（与 $(K,\mathcal{E})$ 兼容的曲率/速度界限）并享有相对于安全可行抽象比较器 $\{z_k^\diamond,v_k^\diamond\}_{k=0}^{T-1}$ 的抽象遗憾 $R_T^{\mathrm{ref}}=o(T)$。假设 $\ell$ 是局部Lipschitz的：存在 $L_\ell>0$ 使得对于所有可容许 $(x,u)$ 和 $(x',u')$，$|\ell(x,u){-}\ell(x',u')|\le L_\ell(\|x{-}x'\|+\|u{-}u'\|)$。在假设4、引理3和引理2的分阶段裕度下，滚动时域律 (22)-(27) 达到

```
R_T^{dyn} = o(T),    R_T^{safe} = o(T)    当 T→∞
```

**证明：**
本节对动态遗憾证明陈述；安全遗憾通过将比较器限制到 $\Pi_{\mathrm{safe}}$ 并使用相同步骤配合机会约束得出。

令 $\{x_k^\star,u_k^\star\}_{k=0}^{T-1}$ 为闭环系统的任何可容许（动态可行）比较器序列，具有与所提控制器相同的界限和机会安全规范。令 $\{z_k,v_k\}$ 为 (22)-(27) 传递的标称序列，$e_k$ 为跟踪误差，因此 $x_k=z_k+e_k$ 且 $u_k=v_k+Ke_k$。瞬时成本差距分解为

```
ℓ(x_k,u_k) - ℓ(x_k^\star,u_k^\star) = (ℓ(z_k+e_k,v_k+Ke_k)-ℓ(z_k,v_k)) + (ℓ(z_k,v_k)-ℓ(x_k^\star,u_k^\star))
    ≤ L_ℓ(‖e_k‖+‖Ke_k‖) + (ℓ(z_k,v_k)-ℓ(x_k^\star,u_k^\star))    (36)
```

通过 $\ell$ 的Lipschitz性质。

求和 (36) 对于 $k=0,\dots,T-1$ 并取期望，

```
∑_{k=0}^{T-1} E[ℓ(x_k,u_k)-ℓ(x_k^\star,u_k^\star)] ≤ L_ℓ(1+‖K‖)∑_{k=0}^{T-1} E‖e_k‖ + ∑_{k=0}^{T-1} E[ℓ(z_k,v_k)-ℓ(x_k^\star,u_k^\star)]
```

通过引理3，$\sum_{k=0}^{T-1}\mathbb{E}\|e_k\| \le C_e T^{1/2}$。因此第一项是 $\mathcal{O}(\sqrt{T})$，即 $o(T)$。

本节将标称差距与抽象遗憾联系起来。令 $\{z_k^{\mathrm{ref}},v_k^{\mathrm{ref}}\}$ 为抽象层产生的参考，$\{z_k^\diamond,v_k^\diamond\}$ 为用于定义 $R_T^{\mathrm{ref}}$ 的抽象比较器。通过 (22)-(27) 在紧缩约束上的最优性以及管道/终端构造，标称成本不比在相同紧缩下直接跟踪抽象参考差：

```
∑_{k=0}^{T-1} E ℓ(z_k,v_k) ≤ ∑_{k=0}^{T-1} E ℓ(z_k^{ref},v_k^{ref}) + ∑_{k=0}^{T-1} Δ_k^{tight}    (37)
```

其中 $\Delta_k^{\mathrm{tight}}\ge 0$ 收集由于以下内容的额外成本： 约束紧缩 $h(z_k)\ge \sigma_{k,t}{+}\eta_{\mathcal{E}}$，限制到庞特里亚金差集 $\mathcal{X}\ominus\mathcal{E}$，以及 终端/管道塑形。因为裕度 $\sigma_{k,t}$ 被校准到固定每窗口风险 $\delta$（引理2）并且管道/终端集是时不变的，序列 $\{\Delta_k^{\mathrm{tight}}\}$ 具有均匀有界期望；因此 $\sum_{k=0}^{T-1}\mathbb{E}\Delta_k^{\mathrm{tight}} = \mathcal{O}(T^0)+\mathcal{O}(T\cdot 0)=o(T)$，其意义是增长率是次线性的。

减去并添加抽象比较器以隔离抽象遗憾：

```
∑_{k=0}^{T-1} E[ℓ(z_k,v_k)-ℓ(x_k^\star,u_k^\star)]
    ≤ ∑_{k=0}^{T-1} E[ℓ(z_k^{ref},v_k^{ref})-ℓ(z_k^\diamond,v_k^\diamond)]
    + ∑_{k=0}^{T-1} E[ℓ(z_k^\diamond,v_k^\diamond)-ℓ(x_k^\star,u_k^\star)]
    + ∑_{k=0}^{T-1} E Δ_k^{tight}    (38)
```

右边的第一个和正是假设下的抽象遗憾 $R_T^{\mathrm{ref}}=o(T)$。第二个和比较抽象比较器 $(z^\diamond,v^\diamond)$ 与任何可容许动态比较器 $(x^\star,u^\star)$。由于 $(z^\diamond,v^\diamond)$ 对抽象层可行并且管道/终端设计将此类参考映射到具有有界跟踪/紧缩开销的动态可容许轨迹，差异可以吸收到紧缩松弛中（或由相同裕度/管道缓冲区的常数倍上界），在期望中至多贡献 $o(T)$。最后一个和根据 (37) 之后的讨论是 $o(T)$。

收集步骤2和3的界限，

```
R_T^{dyn} = ∑_{k=0}^{T-1} E[ℓ(x_k,u_k)-ℓ(x_k^\star,u_k^\star)]
         ≤ O(√T) + o(T) = o(T)
```

将比较器限制到 $\Pi_{\mathrm{safe}}$，满足相同分布鲁棒机会约束（具有裕度）的类。那么相同分解成立并且在固定风险 $\delta$ 下紧缩/松弛项保持 $o(T)$，产生 $R_T^{\mathrm{safe}}=o(T)$。

$o(T)$ 比率背后的常数取决于管道增益 $K$、$(A{+}BK)$ 的Lyapunov对、模糊校准（通过稳态裕度）和局部Lipschitz常数 $L_\ell$，但不取决于 $T$。

定理3关闭了高级学习/规划和低级控制之间的循环。它证明控制架构不抹除抽象级学习保证；相反，它保留它们直到跟踪和安全紧缩引入的次线性损失。

## 6. 实验评估

本节详细说明用于评估所提安全遗憾MPC的实验设计、ROS集成、任务、基线、指标和评估协议。所有组件作为ROS包发布以促进复制（`safe_regret_mpc`、`stl_monitor`、`dr_tightening`、`tube_mpc`）。

### 6.1 平台和ROS集成

**硬件和仿真。** 实验在 (1) 移动操作器（Clearpath Ridgeback上的UR5e）和 (2) 全向仓库机器人（具有机载深度相机的Clearpath Jackal）上进行。仿真使用Gazebo（Fortress）配合ROS Noetic。对于物理试验，机器人运行Ubuntu 20.04/Noetic和`ros_control`；状态估计由VIO（`rtabmap_ros`）和关节编码器提供。

**节点架构。** 图1（补充）显示图。关键节点和主题：
- `/state_estimator`（EKF/UKF）：在`/belief`上发布置信度矩/样本（`geometry_msgs/PoseWithCovarianceStamped` + 自定义消息中的粒子数组）
- `/stl_monitor`：订阅`/belief`、`/map`和规范YAML；在`/stl_robustness`上发布软鲁棒性 $\widetilde{\rho}_k$，在`/stl_grads`上发布谓词梯度
- `/dr_tightening`：订阅残差`/ekf/residuals`；在`/dr_margins`上发布模糊半径 $\epsilon_k$ 和每步裕度 $\{\sigma_{k,t}\}$
- `/tube_mpc`：用OSQP/ECOS求解 (22)-(27)；订阅`/belief`、`/stl_robustness`、`/stl_grads`、`/dr_margins`；发布`/cmd_vel`或关节力`/effort_cmd`
- `/safety_shield`（用于审计的可选CBF过滤器）：使用测量的 $h(x)$ 过滤`/cmd_vel`；记录违规以进行事后分析

STL规范、风险水平 $\delta$、时域 $N$、温度 $\tau$ 和管道参数 $(K,\mathcal{E})$ 通过YAML文件和动态重配置传递。

**时序。** 所有控制器在50 Hz（移动）和200 Hz（操作）运行，求解器预算为8 ms。模糊校准和STL鲁棒性评估在单独线程上以10 Hz运行。时间戳和TF确保一致同步。

### 6.2 任务和STL规范

评估两个代表性任务。

**T1：协作装配（移动操作）。** 机器人必须访问站点 $\{\mathsf{A},\mathsf{B},\mathsf{C}\}$，具有停留时间的拾取/放置零件，并避免由深度相机感知的人类占用区域。STL公式为

```
φ_asm := G_{[0,T]} ( ¬HumanNear ∧
    F_{[0,td]} AtA ∧
    F_{[0,td]}(AtB ∧ G_{[0,Δ]} Grasped) ∧
    F_{[0,td]}(AtC ∧ G_{[0,Δ]} Placed) )
```

谓词从 $z=\psi(x)$ 构造：到站点的接近度、夹持器传感器接触和人类检测器分数。机会安全性用 $\delta\in\{0.05,0.1\}$ 执行。

**T2：部分已知地图中的物流（无遗憾探索/执行）。** 机器人必须在交付物品到时变站点的同时发现被遮挡的过道。STL任务耦合到达-避免与服务率：

```
φ_log := G_{[0,T]} ( ¬Collision ∧
    F_{[0,ts]}(AtS1) ∧
    F_{[0,ts]}(AtS2) ∧
    G_{[0,T]} Battery ≥ b_min )
```

拓扑置信空间规划器提出参考；安全遗憾MPC确保动态可行性、DR安全和遗憾转移。

### 6.3 基线

本节比较：
- **B1 标称STL-MPC**：具有无机会约束的STL成本；假设高斯噪声
- **B2 DR-MPC（无STL）**：$h(x)$ 上的分布鲁棒机会约束，二次跟踪目标
- **B3 CBF屏蔽 + 标称MPC**：二次MPC后跟随机CBF过滤器
- **B4 仅抽象无遗憾规划器**：具有时序逻辑和无遗憾探索的路径规划，由PID控制器跟踪

消融从所提方法中移除单个模块：（无预算）、（无管道）、（无DR，固定裕度）、（无置信空间平均）。

### 6.4 指标

- **满足概率** $\widehat{p}_{\mathrm{sat}}$：具有 $\rho(\varphi;\xi)\ge 0$ 的运行比例；报告Wilson 95%置信区间
- **经验风险** $\widehat{\delta}$：具有 $h(x_t)<0$ 的步骤比例；与目标 $\delta$ 比较
- **动态遗憾** $R_T^{\mathrm{dyn}}/T$ 和 **安全遗憾** $R_T^{\mathrm{safe}}/T$：跨种子平均
- **递归可行性率**：MPC保持可行的时间步比例
- **计算**：每次MPC调用的求解时间（中位数，p90），失败计数
- **跟踪误差和管道占用**：$\|e_t\|_2$ 统计和 $\mathcal{E}$ 的违规（应该为零）
- **校准准确性**：残差分布偏移下的观察违规率与标称 $\delta$ 比较

### 6.5 实现细节

**MPC和优化。** 本实验采用SQP，对 $f$ 和 $h$ 进行线性化；每个子问题根据裕度类型是稀疏SOCP（ECOS）或QP（OSQP）。时域 $N=20$（移动）和 $N=30$（操作）；步长 $\Delta t=0.05$ 秒。二次成本 $\ell(x,u)=\|x-x_{\mathrm{ref}}\|_Q^2+\|u\|_R^2$，其中 $(Q,R)$ 调整以匹配管道反馈 $K$ 的带宽。温度 $\tau=0.1$（归一化谓词），在消融中减少到 $0.05$。

**管道设计。** $K$ 从标称周围线性化动力学的LQR获得。误差集 $\mathcal{E}=\{e:\|e\|_{P}\le r\}$ 使用来自Riccati解的 $P\succ0$；$r$ 被选择使得在标称扰动下 $\Pr(e\in\mathcal{E})\ge 0.99$。本节使用局部Lipschitz界限 $L_h$ 预计算 $\eta_{\mathcal{E}}=L_h \max_{e\in\mathcal{E}}\|e\|$。

**模糊和紧缩。** 残差窗口 $M=200$；Wasserstein半径 $\epsilon_k$ 由经验 $(1-\alpha)$-分位数设置，$\alpha=0.05$，当数据稀缺时有下限 $\epsilon_{\min}$。每步风险 $\delta_t=\delta/(N{+}1)$ 除非另有说明。裕度通过Chebyshev形式（方程 (19)）计算；DR-CVaR用于敏感性研究。

**ROS配置。** 启动文件为每个任务生成图：

```
roslaunch safe_regret_mpc task_assembly.launch
roslaunch safe_regret_mpc task_logistics.launch
```

每个启动文件加载具有以下内容的YAML规范：STL公式（前缀符号）、谓词映射、$(N,\delta,\lambda,\underline r,\tau)$、管道参数 $(K,\mathcal{E})$ 和求解器选项。

### 6.6 协议

**仿真。** 对于每个任务和方法，运行30个种子，具有随机地图遮挡和人类轨迹。噪声模型在情节中偏移以测试模糊适应性（例如增加传感器丢失）。对于物流，抽象规划器每0.5秒使用部分已知地图更新参考。

**物理试验。** 每个方法对每个任务执行三个重复，具有相同初始条件。人类参与者为安全区域遵循脚本路径；实验遵守机构安全指南。

**数据收集。** 本实验记录与决策和评估相关的所有主题（`/belief`、`/dr_margins`、`/stl_robustness`、`/cmd_vel`、`/shield_log`），并发布ROS包和具有提交哈希的元数据。后处理用记录的轨迹重新计算精确鲁棒性 $\rho$。

### 6.7 结果和分析

**安全性和校准。** 跨噪声偏移，所提方法保持 $\widehat{\delta}$ 接近目标 $\delta$，而B1和B4显示低估和约束违规峰值。这确认了引理2导出的模糊校准和裕度的作用。

**任务满足。** 在两个任务上，所提方法的满足率 $\widehat{p}_{\mathrm{sat}}$ 高于B2/B3，特别是在T1的紧凑停留时间窗口和T2的遮挡下。鲁棒性预算防止时域侵蚀：消融（无预算）在截止时间附近显示裕度逐渐损失；使用预算，预测 $\widetilde{\rho}_k$ 保持在基线 $\underline r$ 之上。

**遗憾行为。** 在T2中，抽象规划器在仿真中实现近 $o(T)$ 遗憾；安全遗憾MPC保持次线性动态遗憾，而用PID跟踪（B4）由于不可行参考和饱和显示线性增长。这与定理3和引理3的跟踪界限一致。

**计算和可行性。** 中位数MPC求解时间在热启动下保持在8 ms预算内；可行性率在仿真中保持 $>99.5\%$，在硬件上 $>99\%$。消融（无管道）增加求解时间可变性并在参考激进时产生偶尔不可行性，强调管道/终端集的作用（定理1）。

**案例研究。** 在装配中，当人类短暂遮挡对 $\mathsf{B}$ 的接近时，控制器增加裕度 $\sigma_{k,t}$，延迟接近以维持风险，并恢复以在窗口内满足 $\mathsf{C}$。在物流中，当过道被发现被阻塞时，抽象规划器改变同调类；安全遗憾MPC通过管道重塑参考并维持满足概率。

### 6.8 消融和敏感性

本节研究 (1) 风险分配（均匀与截止时间加权），(2) 模糊半径计划（分位数与固定），(3) 温度 $\tau$，和 (4) DR-CVaR与Chebyshev裕度。截止时间加权风险在固定 $\delta$ 下改善截止时间附近的满足；太小的 $\tau$ 锐化梯度但减慢收敛；DR-CVaR在较高计算成本下产生略少保守性。

### 6.9 可重复性检查清单

- 所有具有`catkin`构建文件、启动脚本和YAML配置的ROS包
- 每个任务的STL规范（`.yaml`）和谓词定义（`.py`/`.cpp`）
- 地图和Gazebo世界、URDF/Xacro模型和控制器增益
- 从ROS包重新生成所有图表和表格的脚本

实验证明安全遗憾MPC将置信空间STL优化、分布鲁棒机会约束安全性和基于管道的可行性集成到实时ROS堆栈中。观察到的风险校准、任务满足保留和地图不确定性下的次线性遗憾与第5节的理论保证一致。

## 7. 结论

本文提出了安全遗憾MPC，一个统一框架，用于在不确定性和部分可观测性下的安全关键机器人控制中集成时序逻辑任务规范、概率安全性和学习保证。通过将STL鲁棒性提升到置信空间、通过数据驱动的分布鲁棒机会约束执行安全性、通过管道/跟踪分解将抽象无遗憾规划下沉到动态可行层，并提供滚动时域中的递归可行性、类ISS稳定性和概率满足证明，该方法弥合了逻辑感知MPC、风险感知控制和置信空间规划之间的差距。在协作装配和负载运输任务上的实证研究验证了理论结果，显示更高的STL满足概率、校准的风险暴露、相对于安全可行基线的次线性遗憾以及实时执行。未来工作将扩展到多机器人协作、部分可观测下的主动感知以及更丰富的逻辑片段（例如嵌套STL）。
